# Vikingos Digitales — Landing Page Flat Design + Navegación

Proyecto HTML + SCSS basado en las últimas maquetas de Vikingos Digitales.

## Estructura

```txt
vikingos-digitales-web-flat-nav/
├─ index.html
├─ css/main.css
├─ scss/main.scss
└─ assets/
   ├─ logo-vikingos.svg
   ├─ backgrounds/
   ├─ projects/
   └─ references/
```

## Uso

Abre `index.html` directamente en el navegador.

## Incluye

- Menú de navegación superior real, sin marco/barra falsa de navegador.
- Secciones: Hero, Servicios, Proyectos, Nosotros/Proceso y Contacto.
- Fondos flat design claros en SVG.
- Assets incluidos dentro del proyecto.
- SCSS editable y CSS listo para usar.
- Responsive mobile con menú hamburguesa.
- Parallax ligero en desktop usando `background-attachment: fixed`.
